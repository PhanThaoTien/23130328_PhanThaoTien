1. Introduction to Java

Java Overview

Java is a widely-used programming language designed for building robust, secure, and platform-independent applications. It follows the principle of "write once, run anywhere", making it a popular choice for developers.

First Java Program Example

Below is a basic "Hello, World!" example in Java:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java World!");
    }
}

This program prints "Hello, Java World!" to the console.
